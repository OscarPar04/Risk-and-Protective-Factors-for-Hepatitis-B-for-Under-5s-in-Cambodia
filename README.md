# Hepatitis B Vaccination Project

This repository contains the analytical code for my final-year undergraduate research project.

## Project Status

This repository is under active development.

The analysis was conducted primarily in Jupyter notebooks.
Code is being progressively cleaned and structured.

## Structure

- analysis notebooks/ – Analysis notebooks
- src/ – Reusable Python functions
- data/ – Notebook used to clean raw data (raw data not included)
- output/ – Generated figures and tables
